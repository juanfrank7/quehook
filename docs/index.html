About

Comana is a service providing summarized information extracted from the GH Archive.

Every hour, the app pulls in the newly available data in the Archive, parses it, generates reports, and stores them for public consumption. Currently, the analysis is basic, but additional statistics will be available over time.

Instructions

Send a GET request to https://932np5mouk.execute-api.us-east-1.amazonaws.com/prod/load. The app will return as JSON object containing URLs for JSON hourly report files that you can use to fetch using additional GET requests (the URLs will be availabe for 15 minutes). Currently, 1,000 files of hourly data will be returned.

Further functionality will be introduced in the future!
